## Introduction

The site you're on right now. Built with Next.js, designed around one goal: make it easy to see what I actually build, not just a list of links.

---

## 🌟 Key Features

- **Custom Projects Section** – showcases real work with code-snippet style thumbnails instead of a generic blog feed.
- **Mixed Visibility Handling** – a project data structure that handles public repos with live code links alongside private and NDA'd work shown as description-only.
- **Resume-First** – a direct resume download front and center.
- **Fully Custom Content** – every project write-up and section is written and structured by me.

---

## 🛠 Tech Stack

- **Framework:** Next.js, React
- **Language:** TypeScript
- **Styling:** Tailwind CSS

---

## 🎯 Why It's Here

A portfolio is itself a fullstack project — routing, content structure, data modeling for projects with varying visibility, and design, all shipped and iterated on like any other product. It earns its place here.
