## Introduction

Odoo ERP is powerful, but it's not fast — pulling a simple report or checking a resource's status usually means clicking through several menus and filters. Orbi wraps all of that in a conversational layer: ask a question in plain language, get a real answer pulled live from the ERP, no menu-hunting required.

---

<FramedImage src="/images/projects/orbi.png" alt="Orbi cover" />

## 🌟 Key Features

- **Conversational Queries** – ask about resources, records, or statuses in plain language and get answers straight from live ERP data.
- **Direct ERP Integration** – connects natively to Odoo's API/ORM layer instead of relying on stale exports or manual syncs.
- **Business-Friendly** – built for non-technical users who know what they need but don't want to learn Odoo's UI to get it.

---

## 🛠 Tech Stack

- **Language:** TypeScript
- **Integration:** Odoo ERP (API/ORM layer)
- **AI:** OpenAI for natural language understanding
